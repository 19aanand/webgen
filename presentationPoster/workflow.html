Vertically Integrated Projects

WebGen is a project team in the Vertically Integrated Program (VIP)​

VIP teams consist of undergraduate students, graduate students, and faculty members

With this interdisciplinary team we needed a workflow that would increase productivity as well as communication among our team​

Observable Notebooks

Observable notebooks and Google documents were used to share our work

Learned how to fetch data from the GDC to show in plotly graphs in our notebooks

Github Repository

We moved code from our Observable notebooks onto our Github repository

Each team member has their own branch to push their work to​

Manage our tasks using the project feature on Github ​

Communication
